css - grid

Flex는 한 방향 레이아웃 시스템이고 (1차원)

Grid는 두 방향(가로-세로) 레이아웃 시스템 (2차원)

  • grid-template-rows는 행(row)의 배치를 결정합니다.
  • grid-template-columns는 열(column)의 배치를 결정합니다.
  • fr(fraction)은 숫자 비율대로 트랙의 크기를 나눕니다. 예를들어 1fr 1fr 1fr은 균일하게 1:1:1 비율인 3개의 column을 만들겠다는 의미입니다.
  • repeat(반복횟수, 반복값)는 반복되는 값을 자동으로 처리할 수 있는 함수입니다.
  • minmax 함수 최솟값과 최댓값을 지정할 수 있는 함수입니다. ex)minmax(100px, auto)
  • auto-fill과 auto-fit은 column의 개수를 미리 정하지 않고 설정된 너비가 허용하는 한 최대한 셀을 채웁니다.
  • row-gap, column-gap, gap 그리드 셀 사이의 간격을 설정합니다. (IE에서는 gap의 대체 속성이 없기 때문에, 처음부터 gap을 사용하지 않고 구조를 설계하는게 편하다.)
  • 각 셀의 영역 지정
    • grid-column-start, grid-column-end, grid-column, grid-row-start, grid-row-end, grid-row
    • -ms-grid-row, -ms-grid-column, -ms-grid-row-span, -ms-grid-column-span
